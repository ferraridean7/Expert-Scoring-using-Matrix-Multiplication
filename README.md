# 🏦 Expert Scoring System using Matrix Multiplication
*Bridging Banking Risk Logic with Machine Learning Scalability*

## 📌 Project Overview
Proyek ini mendemonstrasikan bagaimana logika **Risk Management** perbankan selama 6 tahun dapat ditransformasikan ke dalam sistem penilaian otomatis menggunakan operasi matriks. Fokus utama adalah pada **transparansi keputusan** dan **skalabilitas delivery** dalam menentukan kelayakan kredit.

## 🚀 Business Value
- **Predictable Outcomes**: Menggunakan perkalian matriks untuk memastikan bobot variabel (judgment pakar) diterapkan secara konsisten pada ribuan data nasabah.
- **Efficiency**: Mengotomatisasi proses penilaian manual menjadi komputasi instan.
- **Risk Mitigation**: Menerapkan standarisasi data (Normalization) untuk menjaga akurasi penilaian lintas parameter nasabah yang berbeda.

## 🛠 Tech Stack & Methodology
- **Python (NumPy, Pandas)**: Untuk manipulasi data dan komputasi matriks.
- **Standardization**: Menyamakan skala fitur untuk menghindari bias model.
- **Expert Judgment Weights**: Penentuan bobot variabel berdasarkan pengalaman riil di industri perbankan (Credit Analysis).

## 📂 Project Structure
- `credit_final_cleaned.csv`: Dataset yang telah melalui tahap data wrangling.
- `Expert-Scoring-Model.ipynb`: Dokumentasi logika Python dan visualisasi distribusi skor.
